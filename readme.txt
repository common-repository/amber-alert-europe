== Installation ==
1) Unzip the plugin in your plugins folder and activate the plugin

2) In the settings section,you can enter of numbers of messages ( 1 to 15 ), 

if you want a to display the photo or not , 

and coordinates displayed or not.

and a simple chache ( time can be set )

3) Creating a AMBER Alert Europe page is easy, just add a new Page through your wordpress admin and add following line in that page. [amber]

4) That's it ! Enjoy !
